Readme.htm for MVP Summit Demos

This set of files contains demos and Power Point presentations that were delivered by members of the Microsoft Visual FoxPro (VFP) development team to VFP MVPs during the 2004 MVP Global summit in Seattle, WA USA.

Installing and using MVP Summit Demos

This section contains information about installing and using the MVP Summit Demos. There is no install package per-say. To install these samples, simply download and expanded "MVP_DEMOS.exe". By default, these files are expanded to C:\Europa_MVP_DEMOS ("Europa" is the codename for VFP9). This directory is referred to in the following instructions as the "ExpandDir".

To run the reporting demos, open Visual FoxPro 9 and modify the <ExpandDir>\Reporting\Europa_ReportingChanges.PJX project.

To run the data demos, open Visual FoxPro 9 and modify the <ExpandDir>\Data Enhancements\Europa_DataChanges.PJX project.

Note: Some of the programs in the Data Enhancements directory require a SQL Server installation to run. The connection string for these .PRGs is defined in sqltests.h. When you open Europa_DataChanges.PJX, a Project Hook will kick in and give you the opportunity to change this SQL Server definition. It is recommended you recompile all .PRGs in the project after you do this (the Project Hook code will do this for you if you so desire).

On behalf of the Microsoft VFP DEV Team, enjoy!
