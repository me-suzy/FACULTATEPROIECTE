Visual FoxPro 8.0 has a class called Empty.  This project demonstrates how to use this new class.

Empty.pjx is a project that contains 2 different examples on how to use the Empty class in your applications.

1_EmptyClass_Basic : 
This program will show how to add and remove properties to an instance of an Empty class.

2_EmptyClass_Scatter : 
This program will show how to use the Scatter/Gather and Insert Into functions to copy records from one cursor to another cursor that has a different table structure.
